# Разработка модели прогнозного анализа для предсказания востребованных профессий

## Финальный отчёт

---

## Аннотация

В рамках дипломного проекта разработана система прогнозного анализа для предсказания востребованных профессий на основе данных о вакансиях с платформы HeadHunter.kz. Модель машинного обучения достигла точности **99.15%** при классификации 14 категорий профессий.

---

## 1. Введение

### 1.1 Актуальность

Рынок труда в IT-сфере постоянно развивается, и понимание востребованности различных профессий критически важно для:
- Соискателей при выборе карьерного пути
- HR-специалистов при планировании найма
- Образовательных учреждений при разработке программ

### 1.2 Цель работы

Разработка модели прогнозного анализа для автоматической классификации вакансий по профессиональным категориям на основе:
- Требуемых навыков
- Условий работы
- Текстового описания
- Информации о зарплате

### 1.3 Задачи

1. Сбор и анализ данных с HeadHunter.kz
2. Предобработка и очистка данных
3. Создание признаков (Feature Engineering)
4. Обучение и сравнение моделей ML
5. Оценка качества модели
6. Интерпретация результатов

## 3. Методология

### 3.1 Предобработка данных

**Этапы предобработки:**
1. Парсинг дат и зарплат
2. Очистка HTML-тегов из текста
3. Нормализация категорий опыта работы
4. Стандартизация названий городов
5. Извлечение временных признаков

### 3.2 Feature Engineering

Создано **70 признаков** в 6 категориях:

| Категория | Количество | Метод |
|-----------|------------|-------|
| Навыки | 30 | Бинарное кодирование топ-30 навыков |
| TF-IDF | 20 | Text vectorization описания |
| Категориальные | 4 | Label Encoding |
| Числовые | 4 | Standard Scaling |
| Агрегированные | 4 | Группировка по профессиям и городам |
| Временные | 4 | Извлечение из даты публикации |

### 3.3 Модели машинного обучения

**Сравниваемые модели:**
- Logistic Regression
- Random Forest
- Gradient Boosting
- K-Nearest Neighbors

**Метрики оценки:**
- Accuracy
- Precision (weighted)
- Recall (weighted)
- F1-Score (weighted)

### 4.1 Сравнение моделей

| Модель | Train Acc | Test Acc | Test F1 | CV F1 | Время |

### 4.2 Лучшая модель

**Logistic Regression** выбрана как лучшая модель:
- Высокая точность (99.15%)
- Стабильность (низкая вариативность CV)
- Интерпретируемость коэффициентов
- Быстрое обучение и предсказание
